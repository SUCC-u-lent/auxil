Auxil

Auxil is a web server that serves as an intermediary between clients and the Ollama language model server. It provides a RESTful API for clients to interact with the Ollama server, allowing them to send prompts and receive responses.

API Endpoints

  • GET /: A simple endpoint that returns a welcome message.
  • POST /api/generate: An endpoint that accepts a JSON payload containing a prompt and a model, returning the response from the Ollama server.

Configuration

The server can be configured using the config.json file, which allows you to specify CORS allowed origins, server port, Ollama port, and server address.
